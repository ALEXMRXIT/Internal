# <h1 align="center">Internal Game Engine
Internal Game Engine — это многофункциональный игровой движок для создания 3D игр. Он предоставляет полный набор инструментов, чтобы сосредоточиться на создании игр жанра MMORPG.

![image_2024-12-08_15-48-40](https://github.com/user-attachments/assets/3eca8359-f40e-44b9-9a12-b5fd382da5c8)

Release 1.0.4
1) Выбор различных разрешений экрана
2) MSAA сглаживание X4, X8. X16
3) Фиксированный Update кратно 30 кадров в секунду
4) Обновление состояния за счет системного обновления частоты CPU
5) Парсинг *.obj формата файлов, пока что поддерживается только триангулированная сетка
6) Диффузная текстура, с поддержкой настройки тайлинга и смещения
7) Линейное освещение, окружающий свет (не настраивается от skybox)
8) Поддержка GUI интерфейса, работающий на Microsoft DirectWrite
9) Поддержку 32/64 битной архитектуры
10) Оптимизация загрузки копий вершин для меша
